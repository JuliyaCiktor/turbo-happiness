Команды

  1. git config --global user.name "JuliyaCiktor"(Имя пользователя)
  2. git config --global user.email julia_uk_oleksandrivna@ukr.net (имейл)
  3. git init (инициализация Гит дериктории/локальный гит репозиторий/ локальный/глобальный)
  4. git add index.html (добавить файл)
  5. git add . (добавить все файлы кот существуют в проекте, кот были изменены, или созданы, или удалены. Отследи все изменения кот были локально. С этого момента Гит начал следит за нашими файлами )
  6. git commit -m "Some text" (Уникальная версия кода, имеет уникальный идентификатор, имеет в себе сообщение о версии кода)
  7. git branch -M main (изменить ветку)
  8. git remote add origin https://github.com/JuliyaCiktor/bug-free-happiness.git (подключаемся к удаленному репозиторюю)
  9. git push -u origin main (все изменения передать на удаленный репозиторий)
  • U (flag untrek -не отслеживается Гитом)
  • A (added / добавить файл )
  • М (Modified /Файл изменен )
