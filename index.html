• git config --global user.name "JuliyaCiktor" (Имя пользователя)
  • git config --global user.email julia_uk_oleksandrivna@ukr.net (имейл)
  • git init (инициализация Гит дериктории/локальный гит репозиторий/ локальный/глобальный)
  • git add index.html (добавить файл)
  • git add . (добавить все файлы кот существуют в проекте, кот были изменены, или созданы, или удалены. Отследи все изменения кот были локально. С этого момента Гит начал следит за нашими файлами )
  • git commit -m "Some text" (Уникальная версия кода, имеет уникальный идентификатор, имеет в себе сообщение о версии кода)
  • git branch -M main (изменить ветку)
  • git remote add origin https://github.com/JuliyaCiktor/bug-free-happiness.git (подключаемся к удаленному репозиторюю)
  • git push -u origin main (все изменения передать на удаленный репозиторий)
  • u (flag untrek -не отслеживается Гитом)
  • A (added / добавить файл )
